# MJS TTRPGs

A collection of web-based tools for tabletop RPGs.

## Fates — Character Sheet

An interactive character sheet for the Fates tabletop RPG. Features:

- **Dark theme** with selectable accent colors
- **Multi-character management** — create, load, switch, and delete characters
- **Auto-save** to localStorage with debounced persistence
- **Stress boxes** with clickable number-in-corner toggle
- **Print layout** — two-page output (Identity/Attributes/Stress on page 1, Extras/Stunts on page 2)

### Running

Just open `fates/index.html` in any modern browser — no build step or server required.
